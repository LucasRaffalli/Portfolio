Zenith Network

Bot Discord modulaire / développement, design / en cours
Zenith Network, c’est un bot Discord modulaire avec des sous-bots indépendants pour gérer la modération, les événements, et les interactions entre serveurs.
Rôle: Développement / Design
Stack: Discord.js / MongoDB / TypeScript
Fonctionnalités:
Zenith – Cerveau du Zenith Network
Zeta – Module de Modération Intelligente
Aion – Module de Gestion d'Événements
Echo – Module de Notifications & Webhooks
Luma – Module d’Interactions Communautaires
